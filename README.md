# GustavoInTech Portfolio

Bem-vindo ao meu portfolio pessoal! Aqui mostro meus projetos, habilidades e jornada como desenvolvedor full-stack.

## Tecnologias Usadas

- Frontend: React, Vite, TailwindCSS
- Integrações: EmailJS para formulário de contato
- Deploy: Vercel

## Como Rodar Localmente

1. Clone o repo: `git clone https://github.com/GustavoInTech/gustavointech-portfolio.git`
2. Instale dependências: `npm install`
3. Rode: `npm run dev`
